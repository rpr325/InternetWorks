The Internet Project

At the time of this writing there are currently 4.7 billion internet users across the globe. That is approximately 60 percent of the global population. With 6.5 billion Google searches, 232.5 billion emails sent and 1.8 billion websites It is hard to argue against the utility and perverseness of the internet.

The purpose of this project website is to take a dive into the internet and unlock the topics and concepts of how this ever-growing distributed network we call the Internet works.

Domain Name System

As previously mentioned, the domain name system facilitates accessibility to the internet. The DNS servers is a phone book with listings for all addresses associated with each top-level domain. In order to maintain universal resolvability more than one domain name within associated with a single top level domain can exist.
