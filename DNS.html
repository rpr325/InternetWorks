Domain Name System (DNS)

As previously mentioned, the domain name system facilitates accessibility to the internet. The DNS servers is a phone book with listings for all addresses associated with each top-level domain. In order to maintain universal resolvability more than one domain name within associated with a single top level domain can exist.

For example, you can register the domain name ‘howtheinternetworks’ with top level domains .com, .org, .us… etc. However, the domain name is cannot be registered twice as a .com.

The DNS is a directory spread across the world that contains the list of all purchased domain names on the internet and the corresponding host server IP addresses.

The DNS is able to link users to the appropriate address on the internet for the device with the information the seek.
